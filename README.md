## Manual Setups

### Keyboard

#### Remap CAPS LOCK to ESCAPE

Go to `System Preferences -> Keyboard -> Modifier Keys...` and set Caps Lock Key to `Escape`

![](img/keyboard-keys.png)

### Alfred

In advanced setting page, set preferences folder to `alfred3`.

![](img/alfred-sync.png)
